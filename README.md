# Graph BFS Shortest Path (C++)

## 📌 專案介紹

本專案實作 **無權重無向圖 (Undirected Unweighted Graph)** 的\
**Breadth-First Search (BFS)**，用來計算：

-   每個節點到起點的最短距離
-   每個節點的前驅節點 (Predecessor)
-   起點到終點的實際最短路徑

------------------------------------------------------------------------

## 🧠 功能特色

-   支援 **手動輸入**
-   支援 **檔案輸入 (File I/O)**
-   例外處理 (Exception Handling)
-   使用 RAII 保證檔案安全關閉
-   安全輸入驗證
-   路徑重建 (Path Reconstruction)

------------------------------------------------------------------------

## 📂 專案結構

    .
    ├── main.cpp
    └── function.h

### main.cpp

負責： - 輸入模式選擇 - 起點與終點輸入 - 呼叫 BFS - 顯示結果

### function.h

包含： - `Input()`：手動輸入圖 - `FileInput()`：檔案讀取圖 -
`BFS()`：廣度優先搜尋 - `FindRoute()`：路徑重建

------------------------------------------------------------------------

## 🚀 執行流程

1.  選擇是否使用 File IO
2.  建立圖 (Adjacency List)
3.  輸入起點與終點
4.  執行 BFS
5.  輸出：
    -   每個節點的距離
    -   前驅節點
    -   最短路徑

------------------------------------------------------------------------

## 📥 輸入格式(建立Adjacency List)

### 1️⃣ 手動輸入模式

    numberOfNode numberOfEdge
    u1 v1
    u2 v2
    ...

範例：

    5 4
    0 1
    0 2
    1 3
    3 4

------------------------------------------------------------------------

### 2️⃣ 檔案輸入模式

檔案內容格式與手動輸入相同：

    5 4
    0 1
    0 2
    1 3
    3 4

------------------------------------------------------------------------

## 🧮 演算法說明

### BFS 時間複雜度

    O(V + E)

-   V = 節點數
-   E = 邊數

使用 Queue 逐層擴展節點，保證在無權重圖中得到最短路徑。

------------------------------------------------------------------------

## 🛡 Exception Safety

本專案具有良好的例外安全性：

-   使用 `ifstream` (RAII) 自動關閉檔案
-   輸入錯誤會丟出 `runtime_error` 或 `invalid_argument`
-   主程式使用 `try/catch` 進行錯誤重試
